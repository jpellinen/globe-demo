# globe-demo

This is a small and simple demo of a rotating globe. It features OpenGL, shaders, normal mapping and lens flare effect.

[YouTube](https://www.youtube.com/watch?v=dWgZW0dnMk0)

## Instructions

The demo works by mouse:
* Left button: rotate
* Right button: zoom
